## Overview

This include adds a pre-formatted todo list to your page. This include is typically used to document a list tasks to finish.

The following code:

```
{% raw %}
{% include guidetask content='complete step A |complete step B |complete step C' %}
{% endraw %}
```
produces:

{% include guidetask content='complete step A |complete step B |complete step C' %}
