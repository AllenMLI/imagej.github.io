OtoRing - An ImageJ plugin to aid otolith ring identification.  
The source code is available at https://github.com/tektonia/otoring  
To use the ImageJ Updater use the following url: http://sites.imagej.net/OtoRing/

Example: ![Blue Whiting (*Micromesistius poutassou*) otolith](N51x10_20092013_ipma.jpg "Blue Whiting (Micromesistius poutassou) otolith")  
1 - Select a Region Of Interest using the Line tool

![Opaque and Translucent rings](N51x10_20092013_ipma_1.jpg "Opaque and Translucent rings")  
2 - Execute the OtoRing plugin

![Graph plots intensity by distance](N51x10_20092013_ipma_2.jpg "Graph plots intensity by distance")  
3 - The result is shown

![OtoRing Parameters](N51x10_20092013_ipma_3.jpg "OtoRing Parameters")  
4 - Use the OtoRing Parameters to improve the results
